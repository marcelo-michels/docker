# docker

## Docker Version

  `docker version`

## Listar todos os contaiters que estão rodando 

  `docker ps`

## Listar todos os contaiters que estão parados 

  `docker ps -a`

## Limpar o docker  

  `docker system prune`  
  `docker container prune`

## Inicia um docker compose file  

  `docker-compose up -d`  
 
## Parar um container  

  `docker stop name`  

## Iniciar um container  

  `docker start mongo`  

## remover um container  

  `docker rm mongo`
